Program Features

This extension allows you to apply a blur filter to images and videos on web pages. You can toggle the filter on and off using the following methods:

  • 1. Click the extension's badge icon in the browser toolbar.
  • 2. Use the keyboard shortcut (Ctrl + Shift + Y).
  • 3. Right-click on any element on the page and select "Context" from the context menu.
